# Full Stack Social Media Application
## Description
This is a social media application. Users can share their posts and chat with others.

## Tech Stack
### Frontend
- React.js
- Tailwind css
- Redux

### Backend
- Spring Boot

### Database
- MySQL

## To run the app
### Frontend
```cmd
cd react
npm run dev
```


### Backend
```cmd
cd springboot/demo
mvn spring-boot:run
```
